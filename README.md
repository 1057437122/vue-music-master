#如何使用


##安装
```
// 安装前请先确保已安装node和npm
//安装依赖(可以使用淘宝镜像cnpm)
npm install

//运行
npm run dev

//localhost:6789
```
##手机端访问（要求和电脑端一个局域网）
```
cmd
ipconfig /all
//查看IPv4 地址，默认网关和DNS服务器就可以了，获取本地IP地址
//手机输入 本地IP :6789
```
